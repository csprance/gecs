Priority tier defaults (from cn_net_sync.gd and sync_sender.gd):
- REALTIME: every frame (~60 Hz)
- HIGH:     20 Hz  (gecs_network/sync/high_hz ProjectSetting, default 20), unreliable
- MEDIUM:   10 Hz  (gecs_network/sync/medium_hz ProjectSetting, default 10), reliable
- LOW:       2 Hz  (gecs_network/sync/low_hz ProjectSetting, default 2), reliable  ← NOT 1 Hz
- SPAWN_ONLY: once at spawn (sentinel, not in Priority enum)
- LOCAL:      never synced (sentinel, not in Priority enum)

NetworkSync methods (from network_sync.gd):
- static func attach_to_world(world: World, net_adapter: NetAdapter = null) -> NetworkSync
- func broadcast_full_state() -> void  (server-only)
- func register_send_handler(comp_type_name: String, handler: Callable) -> void
- func register_receive_handler(comp_type_name: String, handler: Callable) -> void
- var reconciliation_interval: float  (get/set; -1.0 = ProjectSettings, 0.0 = disabled)

Authority components (assigned automatically by SpawnManager, never manually):
- CN_LocalAuthority, CN_RemoteEntity, CN_ServerAuthority
- CN_ServerAuthority assigned only when peer_id == 0

Session event components (transient — cleared at START of _process()):
- CN_SessionStarted, CN_SessionEnded, CN_PeerJoined, CN_PeerLeft
- CN_SessionState (persistent): is_connected, is_hosting, peer_count
