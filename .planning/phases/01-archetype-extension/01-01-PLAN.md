<objective>
Extend the Archetype class to handle `rel://` prefixed relationship slot keys in its `component_types` array without breaking any existing component behavior. This is the foundation for structural relationship queries — all subsequent phases depend on Archetype being able to store relationship pair identity alongside component paths.

Purpose: Enable the archetype to distinguish between component resource paths (which get SoA columns) and relationship slot keys (which do not), and to provide a dedicated matching method for relationship pair queries.

Output: Modified `archetype.gd` with relationship slot key support; new test file `test_archetype_relationships.gd` validating all five ARCH requirements.
</objective>

- Script target: "script://<script_resource_path>"

**2. Modify `_init()`** — after sorting `component_types`, populate `relationship_types` and skip column creation for `rel://` keys:

```gdscript
func _init(p_signature: int, p_component_types: Array):
    signature = p_signature
    component_types = p_component_types.duplicate()
    component_types.sort()

    # Separate relationship slot keys from component paths
    for comp_type in component_types:
        if (comp_type as String).begins_with("rel://"):
            relationship_types.append(comp_type)
        else:
            # Only create columns for component paths, NOT relationship slots
            columns[comp_type] = []
```

**3. Modify `add_entity()`** — skip column population for `rel://` keys:
Change the column population loop from iterating `component_types` to iterating only non-rel keys. Replace:

```gdscript

```

This naturally skips `rel://` keys since they were never added to `columns`.

**4. Modify `remove_entity()`** — same change, iterate `columns` keys instead of `component_types` for column swap-remove:
Replace the two loops that iterate `component_types` for column operations:

**5. Modify `clear()`** — iterate `columns` keys instead of `component_types`:

**6. Add `matches_relationship_query()` method** (after `matches_query()`):

```gdscript
func matches_relationship_query(required_rel_keys: Array, excluded_rel_keys: Array) -> bool:
    # Check required: archetype must contain ALL required relationship keys
    for rel_key in required_rel_keys:
        if not relationship_types.has(rel_key):
            return false
    # Check excluded: archetype must contain NONE of excluded relationship keys
    for rel_key in excluded_rel_keys:
        if relationship_types.has(rel_key):
            return false
    return true
```
