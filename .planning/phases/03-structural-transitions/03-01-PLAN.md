This phase makes relationships truly structural — entity archetype membership reflects relationship pairs, enabling Phase 4's query integration to resolve via archetype cache lookup.

Output: Modified entity.gd (batch signals), modified world.gd (relationship archetype moves + REMOVE policy), new test file validating all TRAN requirements.
